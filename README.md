# eslint-config

My personal eslint configurations

## Installation

```
npm install @kyeotic/eslint-config
```

# Use for Browsers

```
"eslintConfig": {
    "extends": [
      "@kyeotic/eslint-config"
    ]
  }
```

# Use in Node

Add this to your `package.json`

```
"eslintConfig": {
    "extends": [
      "@kyeotic/eslint-config/node"
    ]
  }
```

# Recommended Prettier Config

Add this to your `package.json`

```
"prettier": {
    "tabWidth": 2,
    "semi": false,
    "singleQuote": true
  },
```
